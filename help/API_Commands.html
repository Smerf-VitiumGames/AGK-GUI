*Defaults

  All Gadget have a default setting. Setting default colors,sizes and positions for gadget Can save time when creating new gadgets. For example, A if the defaut color for a Api_Commands is set to blue all Api_Commandss will be set with the color blue when created. Defaults are not part of the api language because it is much quicker to edit each type indevidually. The #include. File named "Default can be edited manually."

  ApiSync(ID)   Use instead of regular sync.

  DeleteGadget(ID)   Deletes a gadget and its Children

  GetPointerMoveY()   ` Accepts Integer 1-0, 1 enalbes Api_Commands to be dropped onto other gadgets, 0 false.
