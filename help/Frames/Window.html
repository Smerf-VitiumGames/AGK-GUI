*Defaults

  All Gadget have a default setting. Setting default colors,sizes and positions for gadget Can save time when creating new gadgets. For example, A if the defaut color for a Window is set to blue all Windows will be set with the color blue when created. Defaults are not part of the api language because it is much quicker to edit each type indevidually. The #include. File named "Default can be edited manually."

  SetWindowRelative(id,left_Top_Right_Bottom$,on_off,distanceToTarget,targetID_zeroForWindow,insideOrOutide,resizeBool)
 
  This command sets a gadget relative to another gadget, if movement or resising of the target gadget occurs this gadget will move in
 relation to it with a dsignated spacing. Usage: id of gadget, relitive position ,turn this feature on or off, distance to maintain between its target,
 the id of the target gadget "use zero for the main window", inside or outside inside=0,outside=1, this will bind the gadget to the outide border or the
 inside border of the target. resizebool 0 off 1 on. resizebool resizes the target gadget and does not reposition it. this is useful for rearanging
 menus to make space on a window or expanding an area.
