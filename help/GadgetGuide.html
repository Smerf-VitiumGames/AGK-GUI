Custom Gadgets




Creating a custom gadget has many benefits such as a unique behavior like a flow char or gauge
The Main difference between gadget is only the gadge.kind type. the .kind determins what is is and what it does.
To creat a gadget. Go do the default.agc file and copy an exisiting Default funtion. Change Rename the function to buildMyName()
Set the kind to a number tat has not already been used.


In gadget_build.agc add your kind and function to the addGadgetFunction example: If kind=1 then BuildDefaultButton()
You will then need to create your own include file that will house the commands for your new gadget. creat a file
Command_MYfile.agc. Add this new file as an #include in INIT_Main.agc file in the INIT_API() function. Create a function in your new file called makeGadgetName.

Copy the make function from another COMMAND file and rename it to match your new gadget
